# JavaScript Strings

Strings in JavaScript are used to represent and manipulate textual data.  
A string is a sequence of characters enclosed in quotes.

JavaScript allows strings to be created using:

- Single quotes `' '`
- Double quotes `" "`
- Template literals `` ` ` ``
